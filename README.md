# Starter pack for basic React app

## Setup

```sh
npm install
```

## Development

```sh
npm run dev
```

## Testing

```sh
npm run test        # Watch mode
npm run test:run    # Single run
npm run test:coverage
```

## Production

Build the application in production mode by running:

```sh
npm run build
```

## Code quality

```sh
npm run lint
npm run format
npm run type-check
```
